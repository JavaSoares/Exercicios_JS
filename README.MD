## 📁 Como baixar o projeto

```bash

# Clonar o repositorio
$ git clone https://wwww.github.com/JavaSoares/Exercicios_JS

# Entrar no diretorio 
$ cd Exercicios_JS

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
